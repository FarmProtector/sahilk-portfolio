# Sahil K. | Cloud Security & Infrastructure 

Professional portfolio and technical documentation for cloud-native architectures. This site showcases my 12+ years of experience in Healthcare IT and my transition into AWS Cloud Security and Architecture.

## 🚀 Live Site
Check out the live deployment at: [sahilk.io](https://sahilk.io)

## 🛠 Tech Stack
- **Framework:** [Astro](https://astro.build/) (Static Site Generation)
- **Styling:** Tailwind CSS / DaisyUI
- **Cloud Infrastructure:**
    - **Storage:** Amazon S3
    - **CDN:** Amazon CloudFront (Edge Functions for routing)
    - **Deployment:** AWS CLI & CI/CD Pipeline
- **Certifications:**
    - AWS Certified Solutions Architect Associate (SAA-C03)
    - AWS Certified Security Specialty (SCS-C03) 
    - CompTIA Security+
    - ISC2 Certified in Cybersecurity (CC)

## 📂 Project Focus
This repository contains the source code for my portfolio, including case studies on:
- **IAM Hardening:** Implementing Least Privilege in complex AWS environments.
- **VPC Security:** Architecting resilient network boundaries and inspection points.
- **Infrastructure as Code:** Moving from manual management to automated deployments.
